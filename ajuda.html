Distribuidor


O que é?

O distribuidor permite obter distribuições de frequência conjuntas em relação a um conjunto de expressões de pesquisa.

Exemplos

Distribuição de nomes por género e números

Qual o número de substantivos num corpo? Quantos destes no masculino? Quantos no feminino? O plural é aplicado mais vezes para o masculino ou para o feminino?

?pos=/N/ gen pessnum

Nesta pesquisa temos:

  • ?pos=/N/ é usado para selecionar apenas nomes. O ponto de interrogação antes do nome do atributo indica que não deve ser usado na distribuição, mas apenas como selector;
  • gen é usado para selecionar os géneros dos vários nomes;
  • pessnum é usado para selecionar os números dos vários nomes;

Género dos complementos diretos de... Comer

Considere-se o caso em que se pretende saber que um verbo (neste caso, comer) é mais usado com um complemento direto no feminino ou masculino. Uma expressão de pesquisa possível seria:

lema=/comer/ gen+1

Explicando a expressão de pesquisa:

  • lema=/comer/ significa que procuramos qualquer forma do lema comer;
  • gen+1, significa que queremos analisar géneros, da palavra seguinte (daí o +1). Neste caso não especificamos uma expressão sobre o atributo género, significando que queremos procurar todos os géneros possíveis.

Distribuição de POS por autores

Podemos ver quais as formas gramaticais usadas por cada autor presente no corpo Vercial com:

autor pos

Distribuição de cores por variantes

Podemos comparar o uso das cores em Portugal ou no Brasil no corpo ConDiv, usando uma expressão de pesquisa semelhante a:

?sema=/cor/ variante lema

Que é composta por:

  • ?sema=/cor/ indica que procuramos a função semântica cor. O ponto de interrogação indica que esta propriedade será usada apenas para a deteção de palavras relevantes, mas não será usada na distribuição.
  • variante lema indica que queremos distribuir pelo atributo variante e pelo atributo lema.

Note-se que a ordem é relevante. O exemplo como apresentado distribui primeiro pela variante, e para cada variante os vários lemas usados. Trocando a ordem teríamos, para cada lema, a sua distribuição por variante.
